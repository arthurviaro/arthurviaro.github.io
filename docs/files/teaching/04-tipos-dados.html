Introdução às Funções

  • Em praticamente toda linguagem de programação, temos acesso a funções.
  • Uma função pode ser entendida como uma ação (um “verbo”) que executa alguma tarefa no programa e retorna algum resultado.
  • Toda linguagem já possui um conjunto de funções pré-definidas — ações básicas que o computador já sabe realizar. O programador pode utilizá-las para instruir o computador a executar determinadas operações.
  • No nosso primeiro programa, utilizamos a função print. Essa função exibe algo na tela. Ela não imprime um texto fixo: ela imprime aquilo que você passar como argumento.

Um Pequeno Problema com Aspas

  • Observe como adicionar aspas como parte da sua string é um desafio. print("Olá, "amigo"") não funcionará e o interpretador retornará um erro.

  • Geralmente, existem duas abordagens para corrigir isso. Primeiro, você poderia simplesmente mudar as aspas para aspas simples.

  • Outra abordagem, mais comumente utilizada, seria escrever print("Olá, \"amigo\"").

  • As barras invertidas dizem ao interpretador que o caractere seguinte deve ser tratado como uma aspa dentro da string, evitando um erro do interpretador.

  • Provavelmente, a maneira mais elegante de usar strings seria a seguinte:

  • Observe o f em print(f"Olá, {nome}"). Esse f é um indicador especial para o Python tratar essa string de uma maneira específica, diferente das abordagens anteriores que ilustramos nesta aula.

Fatiamento de Strings (Slicing)

  • O fatiamento de string (slicing) permite que você “corte” uma string para obter uma subseção dela.
  • A sintaxe básica utiliza colchetes e índices: string[início:fim].
    • Índice de Início: Onde o fatiamento começa (incluído).
    • Índice de Fim: Onde o fatiamento termina (excluído).
  • Assim como outras operações de sequência no Python, o fatiamento (slicing) utiliza indexação baseada em zero, onde o primeiro elemento está no índice 0.
  • Pense no índice não como o número do caractere, mas como a distância (ou deslocamento) do início da string.
    • No índice 0, você percorreu 0 caracteres (está no início).
    • No índice 1, você saltou 1 caractere para chegar ao próximo.
  • Dica: Se uma string tem comprimento n, o último índice será sempre n−1.
  • Imagine a string nome = "Python". Os índices são:
Letra P y t h o n
Índice 0 1 2 3 4 5
Negativo -6 -5 -4 -3 -2 -1

  • Você também pode adicionar um terceiro parâmetro: string[início:fim:passo].

  • O passo determina quantos caracteres o Python deve pular.

  • O Python permite contar de trás para frente usando números negativos.

  • O -1 é sempre o último caractere.

  • O slicing é muito usado para limpar dados formatados.

Booleanos

  • Em Python, um booleano representa um de dois valores: True (Verdadeiro) ou False (Falso).
  • Diferente de strings, eles não levam aspas e devem sempre começar com letra maiúscula.
  • True pode ser convertido automaticamente para 1; False para 0.
  • Isso pode ser muito conveniente, por exemplo, ao contar valores verdadeiros ou falsos nas colunas de uma tabela de dados.

Explorando o Python

  • Além de consultar a documentação online, você pode usar funções nativas para descobrir o que cada tipo de dado é capaz de fazer.

Função dir()

  • A função dir funciona como um “mapa”. Ela lista todos os métodos e atributos disponíveis para um objeto ou tipo de dado.
  • Se você quiser saber tudo o que pode fazer com uma string, basta digitar: dir(str).
  • Isso retornará uma lista gigante contendo nomes como strip, title, lower, upper, entre outros. É a forma mais rápida de verificar: “Será que existe um método para isso?”.

Função help()

  • Enquanto o dir diz o que existe, o help explica o que cada coisa faz.
  • Se você esqueceu como usar o strip, por exemplo, pode chamar a ajuda interativa: help(str.strip).
  • O interpretador exibirá uma breve explicação técnica sobre como o método funciona, quais argumentos ele aceita e o que ele retorna.
